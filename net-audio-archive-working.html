Technical Details

How It Works

This archive uses NET Protocol's onchain storage system to permanently store audio files on the Base blockchain.

Storage Flow:

  1. Audio file is read and converted to base64
  2. Metadata is packaged with the audio data
  3. Data is stored via NET Protocol storage contract
  4. Transaction is confirmed on Base blockchain
  5. File becomes accessible via NET Gateway CDN
